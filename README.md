# scrollBar 
###将js文件引入，直接调用即可。
```
  $(function () {
    $("._scroll").scrollBar({
      "speed":50, 
      "barOffTop": 8,
      "barOffBottom": 8,
      "barOffRight": 8,
      "boxWidth": 4,
      "barWidth": 4,
      "barColor": "rgba(255,255,255,0.5)",
      "boxColor": "rgba(167,169,185,1)",
      "isBox":true
    });
  })

```
